Please keep in mind that the program has to calculate the value of the function for every single pixel on your screen and color it accordingly. The more complicated your function is, the longer it will take. Sadly, the only way to avoid this is to reduce the complexity of your function.

The algorithm creates a number of rectangles and then calculates their winding number, essentially calculating the value of the function at 800 point per rectangle. If it is valid it is then split into four smaller rectangles, which means the algorithm does a lot of calculations. Again, if your function is complicated, it might get terminated after 30 seconds.
